# Віртуальний стенд ST841 / ADuC841

Браузерний симулятор навчального стенда ST841 з емуляцією ADuC841, периферійними вузлами, подієвим осцилографом, автономним редактором логічних схем із вкладеними редагованими блоками, тестовими векторами та чотиристанною логікою і вбудованими компіляторами ASM та C51.

C51 backend окремо розподіляє внутрішню RAM і ADuC841 XDATA, підтримує 8/16-бітні скаляри, масиви та packed-структури, вкладені поля, struct-вказівники й реальний XDATA-доступ через `MOVX @DPTR`.

У проєкті залишено лише код і runtime-ресурси симулятора. Готові лабораторні програми, згенеровані `.hex/.ihx/.lst/.rel/.map/.sym` та демонстраційні завантажувачі вилучено. Короткі приклади синтаксису доступні як контекстні підказки безпосередньо в редакторі.

## Запуск

```bash
npm ci
npm run dev
```

## Перевірка

```bash
npm test
npm run build
```

`npm test` перевіряє ASM, C51, препроцесори, автодоповнення, реальне виконання згенерованого HEX у `emu8051.wasm`, реєстратор осцилографа та ядро редактора логічних схем. `npm run build` створює production-збірку.

Матриця підтримки й відомі обмеження наведені в `COMPILER_SUPPORT.md`.


Опис редактора схем, вкладених мікросхем і підключення до ADuC841 наведено в `LOGIC_EDITOR.md`.
